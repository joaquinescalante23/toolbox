# joaquinescalante23/toolbox

Suite de herramientas de ingeniería para automatización de arquitectura, limpieza de deuda técnica, optimización de RAG y validación de razonamiento.

## Skills Disponibles

### 🛠️ fullstack-bridge
Sincronizador de integridad entre Frontend (Next.js) y Backend (FastAPI).
```bash
npx skills add https://github.com/joaquinescalante23/toolbox --skill fullstack-bridge
```

### 💀 ghost-cleanup
Limpieza quirúrgica de archivos y assets huérfanos mediante grafos de referencias.
```bash
npx skills add https://github.com/joaquinescalante23/toolbox --skill ghost-cleanup
```

### 🧠 semantic-context-surgeon
Optimización quirúrgica de ventanas de contexto para RAG híbrido.
```bash
npx skills add https://github.com/joaquinescalante23/toolbox --skill semantic-context-surgeon
```

### 🕸️ graph-rag-density-optimizer
Refinamiento de grafos de conocimiento en Neo4j mediante embeddings.
```bash
npx skills add https://github.com/joaquinescalante23/toolbox --skill graph-rag-density-optimizer
```

### ⚖️ agentic-reasoning-validator
Validación semántica de trazas de razonamiento y detección de alucinaciones.
```bash
npx skills add https://github.com/joaquinescalante23/toolbox --skill agentic-reasoning-validator
```
